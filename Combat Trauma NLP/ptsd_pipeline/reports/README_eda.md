## Generation pipeline

```
python run_pipeline.py --stages 3   # eda.py       — EDA charts + TF-IDF/LR baseline → visuals/*.png
python run_pipeline.py --stages 4   # splitting.py — Stratified split → train/test_dataset.json
python run_pipeline.py --stages 6   # report.py    — Summary docs → slide3_summary.md
```

## Notes
- Hard-negative records (`example_type=hard_negative`) have empty `labels=[]` and are included in all charts.
- Hebrew tick labels rendered via `arabic_reshaper` + `python-bidi` for correct RTL display.
